# üåê Wikipedia Search Application

This is a **dynamic web application** that allows users to search for Wikipedia articles in real-time using the Wikipedia API.  
It displays a list of search results including the **title**, **URL**, and **description** for each article in an elegant and user-friendly interface.

##  Features

   * Search any topic directly from Wikipedia  
   * Real-time results displayed instantly  
   * Beautiful, modern, and responsive UI (Dark theme)  
   * Uses the **Wikipedia REST API** for fetching results  
   * Built with **HTML, CSS, JavaScript**, and **Bootstrap**


 ## Technologies Used

| Technology | Purpose |
|-------------|----------|
| **HTML5** | Structure of the app |
| **CSS3 (Modern Gradient & Glassmorphism)** | Styling and responsiveness |
| **JavaScript (ES6)** | Dynamic content and API integration |
| **Bootstrap 4** | Prebuilt responsive layout components |
| **Wikipedia Search API** | Fetching live data |

## How It Works

1. User types a keyword in the input box and presses **Enter**  
2. A loader spinner appears while data is fetched  
3. Wikipedia API returns search results in JSON format  
4. Each result is dynamically displayed with title, link, and short description
